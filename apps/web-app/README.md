This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started
### Setup
1. install all packages `npm install`

First, run the development server:

```bash
npm run dev
# or
yarn dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Below is a brief overview of the structure:

```
nextjs/
├── assets/         # All assets live here
│   └── images/     ## All image assets
├── components/     # Shared components and utilities
│   ├── ui/         ## UI components
│   └── utils/      ## Shared utility functions
├── pages/          # Next.js pages/routing (please read the docs (https://nextjs.org/docs/routing/introduction) before editing)
├── public/         # Static files at the web root
└── styles/         # All CSS styling lives here
```
